USEFUL VaME CONSOLE COMMANDS FOR TESTING!

Open the console with / or \
Use these commands without the quotes
Press enter with nothing typed to dismiss the console

"script level2" - skips to level 2. only works at the very beginning of the game in the white room.

"save load" - loads a save.

"display selectobject" - displays when you're near an object. try it out... if you think it'd be a good feature for the final product let me know!

"display mplayerlocation" - displays location co-ordinates of main player.

"display frames" - display fps and how many frames have been pushed.

"display techspecs" - display all the tech specs.

"display off" - turn off the info display.

"say whatever" - say something! whatever you want!

"system quit" - quits the game.

"system now" - quits the game immediately (skips unloading and other quit processes.

"system hang" - same as system now but keeps the window open until a key is pressed.

"system restart" - restarts the game.
